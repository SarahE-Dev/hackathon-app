## Project Overview

**CodeArena** is a production-ready assessment and hackathon platform built with:
- **Frontend**: Next.js 14 with Tailwind CSS (neon cyberpunk theme)
- **Backend**: Express.js with TypeScript
- **Database**: MongoDB
- **Cache/Queue**: Redis
- **Deployment**: Docker Compose
